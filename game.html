KARTS

As our final project, we decided to create an agumented reality game on an iphone. Our goal was to create a functional race game using what we have learned within these past weeks. We decided to use Unity, to create the game, characters,the race track, etc. Through these last two weeks, we encountered many errors but at the end of the day we managed to solve most of it with the help of our peers and teachers.One of the main challenges we faced was trying to get the AR to funcproperly in Unity. Another challenge we faced was in the wesite with the HTML and CSS codes. Our end result was a fully functional website and a simple website. In the future we would like to make the game more interactive and actually add a kart. Lastly, we would like to give special thanks to Christine Meyhard and Justin Restaino for helping us create our game.
